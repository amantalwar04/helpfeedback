Beginning a Topic

With the exception of glossary topics, you must include a title and prolog section before you begin the body of the topic. In addition, you can optionally include a short description of the topic. The following sections provide guidelines for these common elements.

Note:

When creating a new topic, always start with the Microchip template for the corresponding information type, if one is available. If you copy another topic, you could inadvertently duplicate element IDs and you risk overlooking elements that you might need for the new topic that were removed from the topic you copied.
