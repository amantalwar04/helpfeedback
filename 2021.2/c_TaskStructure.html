Structure

Structure your task topics as follows, keeping each element in the order shown. If you choose to omit an optional element, go to the next element in the structure.
  • <task> (required)
  • <title> (required)
  • <shortdesc> (required)
  • <prolog> (required)
  • <taskbody> (required)
    • <prereq> (optional)
    • <context> (optional)
    • <steps> (required)
    • <result> (optional)
    • <example> (optional)
    • <postreq> (optional)

This structure applies to all categories of task topics.
