2.2.5.16 X Libraries are Not in the LD_LIBRARY_PATH (Libero)

(Ask a Question)

Description: Libero has a dependency on the X Libraries, but cannot find them.

Solution: Install the libXft package:

% sudo yum install libXft.i686
